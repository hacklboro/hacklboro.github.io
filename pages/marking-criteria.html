Marking Criteria
Criteria General description 0 or 1 2 or 3 4 or 5 Points
Problem Is the problem identified relevant to HackLboro 2021's theme? The problem doesn't address the theme. The problem partially addresses the theme. The problem clearly addresses the theme. -
Business Value How significant would the final system be in the marketplace? There's no market for the system. There is little demand for the system. The system has a clear market. The system has a market and has the potentially to be expanded to increase the demand for it. -
System Functionality Does the system work and does its functionality address the problem? The system is not fully functional. Most of the system's critical functionality is working. All critical functionality of the system is working. -
Presentation Did the presentation of the system describe its core functionalities and how these relate to the problem? Was it clear? The core functionalities of the system weren't addressed. The core functionalities of the system were addressed. The relationship with the problem was mentioned. The presentation was generally clear. The core functionalities of the system were addressed and a clear relationship with the problem was established. The presentation was very clear. -
