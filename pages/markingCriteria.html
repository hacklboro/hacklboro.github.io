Marking Criteria

Criteria General description 0 - 1 points 2 - 3 points 4 - 5 points Weight
Problem Is the problem identified relevant to HackLboro 2021's theme? Is the idea behind the system creative? The problem doesn't address the theme. The idea lacks creativity. The problem partially addresses the theme. The idea is somewhat creative. The problem clearly addresses the theme. The idea is very creative. 4
Business Value How significant would the final system be in the current and future marketplace? Do the participants identify a possible marketing strategy for the system? The system has no clear market. The participants do not mention a marketing strategy for the system. The system has a clear current market. The relevance of the system in the future market is not addressed. A marketing strategy is broadly mentioned. The system has a clear market. The system has the potential to still be relevant in the years to come. A strong marketing strategy is addressed. 6
System Functionality Does the system work and does its functionality address the problem identified? Is the code understandable and generally well-structured? The system is not fully functional. The system's working functionality does not address the problem identified. The code is not understandable. Most of the system's critical functionality is working and addresses the problem identified. The code is generally well-structured and understandable. All critical functionality of the system is working and addresses the problem identified. All the code is well-structured and understandable. 6
Presentation Did the presentation of the system describe its core functionalities and how these relate to the problem? Was it clear? Was team effort evident? The core functionalities of the system weren't addressed. The presentation wasn't clear. Team effort was not demonstrated. The core functionalities of the system were addressed. The relationship with the problem was mentioned. The presentation was generally clear. Team effort was evident. The core functionalities of the system were addressed and a clear relationship with the problem was established. The presentation was very clear. Team effort was evident. 4
